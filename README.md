# redis<br/>
# 安裝方法<br/>
### 到https://redis.io/download 找到stable的下載連結<br/>
### 複製<br/>
### 進入linux<br/>
### cd /usr/local/src<br/>
### wget 貼上剛剛複製的連結<br/>
### 解壓原碼並進入目錄<br/>
### tar zxvf redis-........<br/>
### cd redis-......<br/>
### make<br/>
### 等待編譯......<br/>
# 如果編譯中出現時間錯誤<br/>
### 解決方法: date -s 'yyy-mm-dd hh:mm:ss' 重寫時間<br/>
### 再使用: clock-w 寫入 cmos<br/>
